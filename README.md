# Learning-German


| English        | German          | Pronunciation     |
|----------------|-----------------|-------------------|
| Hello          | Hallo           | hah-loh           |
| How are you?   | Wie geht’s?     | vee gates         |
| I’m fine       | Mir geht’s gut  | meer gates goot   |
| Thank you      | Danke           | dahn-kuh          |
| Please         | Bitte           | bit-tuh           |
| Yes / No       | Ja / Nein       | yah / nine        |
| My name is...  | Ich heiße...    | ikh hye-suh       |
